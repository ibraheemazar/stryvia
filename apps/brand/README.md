# Brand Portal

Brand-side application where advertisers and brand managers browse integration opportunities, review valuations produced by the engine, and track placements across productions.

**Stack (planned):** Next.js (App Router) + React + TypeScript + Tailwind.

**Currently a placeholder** — this app ships in **Phase 2**. The folder exists now so workspace tooling and shared packages are wired up from day one. See `docs/architecture/` for the phased roadmap.
