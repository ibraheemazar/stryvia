# @stryvia/i18n

Translation catalogues and i18n utilities for the STRYVIA apps. STRYVIA ships in **Arabic and English** at launch, with full RTL support, locale-aware number/currency formatting, and shared translation keys across apps.

**Stack (planned):** A lightweight wrapper around `next-intl` (or equivalent), plus JSON message catalogues per locale.

**Currently a placeholder** — translation catalogues and helpers are added once UI copy starts landing. See `docs/architecture/` for the staged build plan.
