# @stryvia/ui

Shared React component library used by all STRYVIA frontend apps (Producer Workspace, Brand Portal, Admin Console). Houses the design system primitives, form controls, layout shells, charts, and brand-aligned theming.

**Stack (planned):** React + TypeScript + Tailwind, with Radix UI primitives and a Storybook for component preview.

**Currently a placeholder** — components are added incrementally as the apps need them. See `docs/architecture/` for the staged build plan.
